# ProyectoColaborativo-ServidorCorreo
Proyecto colaborativo de envío de correos a un destinatario elegido mediante un formulario.
Los archivos están almacenados en una base de datos y la aplicación recorre dichos archivos (postales)
Con la ayuda del sevidor axigen, juntamos el destinatario, el emisor y el mensaje del correo junto con el archivo (imagen) seleccionada.

Antes de descargar, cabe recalcar que hemos descargado axigen en un Ubuntu 22.04, para ello descargamos la ultima versión en un .zip desde la página
axigen.com es importante configurar los puertos 80 y 443 ya que si posees un apache server corriendo, dichos puertos se solapan y colapsa el servidor
Es posible que si pasas tiempo sin usar el servidor, se te pase la licencia gratuita de 1 semana.
